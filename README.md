# 🚀 Desafio Técnico – Desenvolvedor Front-End Júnior

**CDT Software**

---

## 📌 Descrição

Este teste tem como objetivo avaliar conhecimentos básicos em:

* React
* TypeScript
* Organização de código
* Consumo de API
* Criação de interfaces simples

A proposta é desenvolver uma aplicação utilizando **React + Vite + TypeScript**, consumindo uma API pública e exibindo dados de forma interativa.

---

## 🧩 Funcionalidades

A aplicação deve:

* 📄 Listar usuários consumindo o endpoint:
  https://jsonplaceholder.typicode.com/users

* 👤 Exibir para cada usuário:

  * Nome
  * Email

* 🔍 Implementar um campo de busca:

  * Filtrar usuários pelo nome em tempo real

* 🖱️ Exibir detalhes ao clicar em um usuário:

  * Nome completo
  * Email
  * Telefone
  * Empresa
  * Cidade
  * Pode ser em **modal** ou **página separada**

* ⚠️ Tratar estados da aplicação:

  * Loading (carregando dados)
  * Erro (falha na requisição)

---

## 🏗️ Requisitos Técnicos

* Utilizar **React com Vite**
* Utilizar **TypeScript**
* Código deve ser:

  * Organizado
  * Componentizado
  * Tipado corretamente

---

## 📦 Entrega

* Disponibilizar o projeto em um repositório:

  * GitHub, GitLab ou Bitbucket

* O projeto deve rodar com:

```bash
npm install
npm run dev
```
